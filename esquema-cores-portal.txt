Considerações:
- Sempre verificar contraste para acessibilidade (WCAG)
- Manter consistência de aplicação das cores
- Usar espaço em branco para clareza
- Feedback visual: verde para sucesso, vermelho para erro, amarelo para aviso
- Paleta deve reforçar a identidade da marca

Resumo: Use azul profundo (#1e3a8a) como base, tons análogos para harmonia, laranja/amarelo para destaque, verdes para ações principais, cinzas para áreas neutras e branco para clareza.
